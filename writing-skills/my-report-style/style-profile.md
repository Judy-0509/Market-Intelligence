# 문체 프로파일 (Style Profile)

> 이 파일은 `style-extractor` 스킬이 채우고, `my-report-style` 스킬이 읽는다.
> 아래 값은 전부 템플릿 상태(`(미학습)`)다. 아직 학습된 내용이 없다는 뜻이며, 이
> 상태에서 `my-report-style`을 실행하면 스킬이 먼저 `style-extractor` 실행을
> 안내하고 멈춘다.
>
> **보안 참고:** 이 파일에는 문체 규칙과 15자 이내 짧은 예시 구만 들어간다. 원문
> 문장이나 문서 내용 전체는 절대 들어가지 않는다.
